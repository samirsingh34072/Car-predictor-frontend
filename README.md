# Car Predictor Frontend

React + Tailwind CSS frontend for the Flask car price prediction API.

## Backend

Start the backend from:

```text
C:\Users\91620\OneDrive\Desktop\Car-Prediction-System\Car-Price-Prediction-master
```

```bash
python app.py
```

The frontend expects:

```text
http://127.0.0.1:5000
```

## Frontend

Install dependencies:

```bash
npm install
```

Run the React app:

```bash
npm run dev
```

Open the URL printed by Vite, usually:

```text
http://localhost:5173
```
